

# UGC Video Creator Platform

## Dashboard
- Clean overview showing recent projects, quick-create button, and stats (scripts generated, products added)

## Product Management
- **Manual entry**: Add product name, description, key features, benefits, target audience, niche category, and upload product images
- **URL import**: Paste a product/landing page URL → AI extracts product details automatically (name, features, benefits, images)
- Product library organized by niche/category with search and filters

## AI Script Generator
- Select a product from your library
- Choose video style: **Product Review**, **Unboxing**, **Tutorial/How-to**, or **Testimonial**
- Choose tone: casual, enthusiastic, professional, relatable
- Choose duration: 15s, 30s, 60s, 90s
- AI generates a complete UGC-style script with:
  - **Hook** (attention-grabbing opening)
  - **Body** (authentic product experience narrative)
  - **CTA** (call to action)
  - **Shot-by-shot storyboard** with visual directions
- Regenerate or edit any section individually
- Save scripts to your library

## Script Library
- Browse all generated scripts by product, niche, or style
- Copy scripts, export as PDF, or mark favorites
- Version history for each script

## Video Generation (Future-Ready)
- Placeholder section for connecting HeyGen/Synthesia API key later
